# DashEngine

...is a very simple Chess Engine made with JavaScript and ElectronJS.

Inspiration: "Coding Adventure - Chess AI" on YT by Sebastian Lauge

Features:

-   [x] Player vs AI
-   [x] Alpha-Beta Pruning
-   [x] Electron Desktop App
-   [x] Stats - Settings
-   [x] View last positions
-   [ ] Transposition Table
-   [ ] Iterative Deepening
-   [ ] AI vs AI
